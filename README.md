Movie Catalog
=============

You can catalog your movies.

Prerequisites

Git
Drush

Installation

  Go to a temporary folder
  $ cd /tmp

  Copy and paste this into your shell to install everything:
  $ curl -L -s http://goo.gl/??? | bash

  Go inside the folder
  $ cd movie_catalog
  
  Run install i.e.
  $ ./install.sh /Users/marcelo/Sites/movie_catalog
